# AI Engineer Portfolio

A sleek, responsive portfolio website built for an AI Engineer. 
The site features a dynamic "token stream" canvas background and modern glassmorphism design.

## Features
- **Dynamic Background:** An animated token stream built using HTML5 Canvas.
- **Glassmorphism Design:** Modern UI styling using raw CSS.
- **Responsive:** Fully responsive on mobile, tablet, and desktop.
- **Single Page Architecture:** Smooth scrolling navigation.

## Quick Start
To view the site locally, simply open `index.html` in any modern web browser. No build steps are required.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript
